---
title: "Detalhe da conversa"
openapi: "GET /api/v1/conversations/{id}"
---

Retorna uma única conversa do workspace pelo seu `id`.

**Scope exigido:** `read:conversations`.

Se a conversa não existir no workspace da chave, a resposta é `404 not_found`.

### Exemplo

```bash
curl https://api.highermind.com.br/api/v1/conversations/5e2c0a1b-1111-2222-3333-444455556666 \
  -H "Authorization: Bearer hm_sua_chave_aqui"
```
